# Oh My Posh

Fish currently initializes Oh My Posh with `torii-zayed.omp.json`, which is
included beside this file. The active executable is Oh My Posh `29.31.1`.

The active executable is a host-specific x86-64 binary from `~/.local/bin` and
is not vendored here because it would make the installer non-portable across
Gentoo, Arch, Fedora, NixOS, and other architectures. The future package layer
must provide Oh My Posh `29.31.1` or a deliberate compatible version.
