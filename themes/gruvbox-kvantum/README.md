# Gruvbox Kvantum theme

The active configuration selects `gruvbox-kvantum`. Its local theme files
are intentionally not bundled yet because the examined payload contains no
redistribution license or notice. Do not substitute `kvantum-dark` silently.

Before bundling the exact active `gruvbox-kvantum.kvconfig` and SVG, confirm
permission or obtain a compatible upstream license and record it in
`docs/asset-provenance.md`.
